# Space Invaders - Java
Videojuego 2D clásico desarrollado en Java con programación orientada a objetos.

## Tecnologías
- Java (JDK 8+)
- AWT / Swing
- VS Code / NetBeans

## Características
- Oleadas de enemigos con dificultad incremental
- Sistema de disparos y detección de colisiones
- Contador de puntuación
- Reinicio sin cerrar la aplicación (tecla R)

## Cómo ejecutar
1. Clona el repositorio
2. Abre el proyecto en tu IDE
3. Ejecuta la clase Main
